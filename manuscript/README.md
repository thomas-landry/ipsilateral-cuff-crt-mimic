# Manuscript — ipsilateral-cuff-crt-mimic

## Files

- `tex/manuscript.tex` — the LaTeX source. Compiles to a single PDF.
- `references.bib` — Vancouver-style BibTeX. Verified against PubMed 2026-05-27.
- `captions.md` — figure captions in their submission form (for journal copy-edit).

## Build

From this directory:

```bash
cd tex
pdflatex manuscript.tex
bibtex manuscript
pdflatex manuscript.tex
pdflatex manuscript.tex
```

Or with `latexmk` (recommended):

```bash
cd tex
latexmk -pdf manuscript.tex
```

Figures live at `../figures/fig{01,02,03}*.{pdf,png}` and are regenerated by `uv run python scripts/50_figures.py --which all` from the repo root.
